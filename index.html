Git Commands

  • git clone
    • Clones a repository to the current directory
  • git status
    • shows you what's different between your version and the version online
  • git add (filename or *)
    • adds your updates files to items that need to be commited
  • git commit -m "message"
    • commits changes made before the push
  • git push
    • pushes local changes to github
  • git pull
    • pulls down from the github cloud. Only works when you're in the right directory
